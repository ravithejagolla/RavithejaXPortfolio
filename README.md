# Ravitheja - Full Stack Developer Portfolio

## Introduction
This is a personal developer portfolio built to showcase my skills, projects, and contact information. Designed with a sleek glassmorphism aesthetic, smooth scroll animations, and a fully responsive layout, the portfolio provides a premium and dynamic user experience. It highlights my expertise as a backend-heavy full stack developer.

## Features
- **Responsive Design**: Adapts beautifully across mobile, tablet, and desktop screens.
- **Glassmorphism UI**: Premium frosted glass aesthetic with subtle gradients and drop shadows.
- **Smooth Scroll Animations**: Elements smoothly reveal themselves using custom Intersection Observers and Framer Motion.
- **Dynamic Projects Showcase**: Highlights best work with direct links to live deployments and GitHub repositories.
- **Downloadable Resume**: Direct integration with Google Drive for easy access to my resume.
- **Interactive Contact Section**: A styled contact form with direct links to my professional email, LinkedIn, and GitHub.
- **Custom Assets**: Uses CSS custom properties for consistent theming and a fully customized SVG favicon.

## Design Decisions & Assumptions
- **React + Vite**: Chosen for lightning-fast development, Hot Module Replacement (HMR), and highly optimized production builds.
- **Vanilla CSS Customization**: Elected to use Vanilla CSS instead of a framework like Tailwind to allow granular, custom control over complex glassmorphism effects and gradient animations.
- **Framer Motion**: Integrated for high-quality, physics-based micro-animations that give the site a lively, state-of-the-art feel.
- **Lucide React**: Used for clean, minimal, and consistent iconography.
- **Single Page Architecture**: Utilizing semantic sections and ID-based routing (`#about`, `#projects`) provides an uninterrupted, app-like user experience.

## Installation & Getting Started
Follow these steps to set up the portfolio locally:

```bash
# Clone the repository
git clone https://github.com/ravithejagolla/RavithejaXPortfolio.git

# Navigate to frontend directory
cd RavithejaXPortfolio/frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Usage
### Running the Development Server
```bash
npm run dev
```
Open your browser and navigate to the local server address provided in the terminal (usually `http://localhost:5173`).
